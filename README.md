# YKiosk 소개
YKiosk는 간단한 바자회같이 주문 시스템이 필요한 곳에서 사용할 수 있도록 만드는 키오스크 같은 서비스다.<br>
웹/앱 둘 다 작동하며 앱은 우선 안드로이드 환경으로 먼저 개발할 계획이다.<br>
또한 앱에서는 블루투스 영수증 프린터를 연결하여 주문 시 번호표, 영수증을 출력할 수 있게 만들것이다.<br>
카카오 소셜로그인을 이용해 회원가입이 가능하며 본인의 가게를 만들고 그 안에서 메뉴를 등록하여 주문을 받을 수 있다.<br>
이 키오스크는 결제시스템이 없고 고객과 매장 사이 주문을 도와주며 주문 통계, 메뉴 관리 등을 제공하는 것을 목표로 하는 서비스다.
